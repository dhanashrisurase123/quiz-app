# Online Quiz Application

## Features
- Multiple choice quiz
- Countdown timer
- Auto-submit when timer ends
- Results screen with score

## Tech Stack
- React
- JavaScript
- HTML
- CSS

## Setup Instructions

1. Clone the repository
   git clone https://github.com/dhanashrisurase123/quiz-app.git

2. Navigate to project folder
   cd quiz-app

3. Install dependencies
   npm install

4. Run the app
   npm start

5. Open in browser
   http://localhost:3000
